Πολιτική Απορρήτου & GDPR

Τελευταία ενημέρωση: {{ current_date or 'Νοέμβριος 2025' }}

1. Εισαγωγή

Η προστασία των προσωπικών σας δεδομένων είναι υψίστης σημασίας για εμάς. Η παρούσα πολιτική περιγράφει πώς συλλέγουμε, χρησιμοποιούμε, αποθηκεύουμε και προστατεύουμε τα προσωπικά σας δεδομένα σύμφωνα με τον Γενικό Κανονισμό Προστασίας Δεδομένων (GDPR - EU 2016/679).

2. Υπεύθυνος Επεξεργασίας

Υπεύθυνος επεξεργασίας των προσωπικών δεδομένων είναι το ScanmyData.

3. Δεδομένα που Συλλέγουμε

3.1 Δεδομένα Εγγραφής

  • Email address
  • Όνομα χρήστη (username)
  • Κωδικός πρόσβασης (κρυπτογραφημένος)
  • Ημερομηνία εγγραφής

3.2 Στοιχεία Σύνδεσης myDATA

  • User ID ΑΑΔΕ
  • Subscription Key (κρυπτογραφημένο)
  • ΑΦΜ πελατών

3.3 Δεδομένα Χρήσης

  • IP addresses
  • Ημερομηνία και ώρα σύνδεσης
  • Ενέργειες χρηστών (login, logout, downloads, uploads, exports)
  • Browser και λειτουργικό σύστημα

3.4 Επιχειρηματικά Δεδομένα

  • Παραστατικά (τιμολόγια, αποδείξεις)
  • Πελατολόγιο (ΑΦΜ, επωνυμίες)
  • Κατηγοριοποιήσεις εξόδων
  • Αρχεία εξαγωγής (Excel, γέφυρες)

4. Σκοπός Επεξεργασίας

Επεξεργαζόμαστε τα δεδομένα σας για τους εξής σκοπούς:

  • Παροχή Υπηρεσίας: Ανάκτηση και επεξεργασία παραστατικών myDATA
  • Ασφάλεια: Αναγνώριση χρήστη και προστασία λογαριασμού
  • Βελτίωση: Ανάλυση χρήσης και βελτιστοποίηση της πλατφόρμας
  • Υποστήριξη: Επίλυση τεχνικών προβλημάτων
  • Νομικές Υποχρεώσεις: Τήρηση logs για audit purposes

5. Νομική Βάση Επεξεργασίας

Η επεξεργασία των δεδομένων βασίζεται σε:

  • Συγκατάθεση: Με την εγγραφή σας αποδέχεστε την επεξεργασία
  • Εκτέλεση Σύμβασης: Για την παροχή της υπηρεσίας
  • Νόμιμο Συμφέρον: Ασφάλεια και βελτίωση υπηρεσιών
  • Νομικές Υποχρεώσεις: Τήρηση αρχείων και logs

6. Αποθήκευση Δεδομένων

Τα δεδομένα αποθηκεύονται:

  • Firebase (Google Cloud): Authentication & Realtime Database
  • Τοπικοί Διακομιστές: Αρχεία παραστατικών και εξαγωγών
  • Κρυπτογράφηση: Ευαίσθητα δεδομένα (passwords, API keys) κρυπτογραφούνται

7. Κοινοποίηση σε Τρίτους

ΔΕΝ πουλάμε, ενοικιάζουμε ή μοιραζόμαστε τα προσωπικά σας δεδομένα με τρίτους, εκτός από τις ακόλουθες περιπτώσεις:

  • ΑΑΔΕ myDATA: Για ανάκτηση παραστατικών (μέσω API)
  • Firebase/Google Cloud: Υποδομή αποθήκευσης (GDPR compliant)
  • Νομικές Υποχρεώσεις: Αν απαιτηθεί από δικαστική απόφαση

8. Διατήρηση Δεδομένων

Διατηρούμε τα δεδομένα σας:

  • Λογαριασμός Ενεργός: Όσο χρησιμοποιείτε την υπηρεσία
  • Μετά τον Τερματισμό: 6-12 μήνες για logs (audit trail)
  • Backups: Μέχρι 30 ημέρες μετά τη διαγραφή

9. Τα Δικαιώματά Σας (GDPR)

Έχετε το δικαίωμα να:

  • Πρόσβαση: Να ζητήσετε αντίγραφο των δεδομένων σας
  • Διόρθωση: Να διορθώσετε ανακριβή δεδομένα
  • Διαγραφή: Να ζητήσετε τη διαγραφή των δεδομένων σας ("right to be forgotten")
  • Περιορισμός: Να περιορίσετε την επεξεργασία
  • Φορητότητα: Να λάβετε τα δεδομένα σε δομημένη μορφή
  • Εναντίωση: Να αντιταχθείτε στην επεξεργασία
  • Ανάκληση Συγκατάθεσης: Να ανακαλέσετε τη συγκατάθεσή σας

Για να ασκήσετε τα δικαιώματά σας, επικοινωνήστε μαζί μας στο privacy@scanmydata.gr

10. Cookies

Χρησιμοποιούμε cookies για:

  • Απαραίτητα: Session cookies για authentication (δεν μπορούν να απενεργοποιηθούν)
  • Λειτουργικά: Αποθήκευση προτιμήσεων (dark mode, επιλεγμένη ομάδα)
  • Ανάλυσης: Για βελτίωση της εμπειρίας χρήστη (μπορούν να απενεργοποιηθούν)

Μπορείτε να διαχειριστείτε τα cookies από τις ρυθμίσεις του browser σας.

11. Ασφάλεια

Εφαρμόζουμε μέτρα ασφαλείας όπως:

  • SSL/TLS encryption για όλες τις επικοινωνίες
  • Κρυπτογράφηση ευαίσθητων δεδομένων σε ηρεμία (at rest)
  • Έλεγχος πρόσβασης με authentication/authorization
  • Τακτικά backups και disaster recovery plans
  • Logging και monitoring για ανίχνευση παραβιάσεων

12. Παραβίαση Δεδομένων

Σε περίπτωση παραβίασης δεδομένων θα ενημερώσουμε:

  • Την Αρχή Προστασίας Δεδομένων Προσωπικού Χαρακτήρα εντός 72 ωρών
  • Τους επηρεαζόμενους χρήστες άμεσα

13. Ανήλικοι

Η υπηρεσία απευθύνεται σε χρήστες άνω των 18 ετών. Δεν συλλέγουμε εν γνώσει μας δεδομένα ανηλίκων.

14. Αλλαγές στην Πολιτική

Ενδέχεται να ενημερώσουμε την πολιτική απορρήτου. Θα σας ειδοποιήσουμε για σημαντικές αλλαγές μέσω email.

15. Επικοινωνία
