Email Configuration
Select the email service to use for sending emails.
SMTP: Traditional email using SMTP server (configured via environment variables)
Resend: Modern API-based email service (requires RESEND_API_KEY in .env)
Mailgun: HTTP API email service - works on Render free tier (requires MAILGUN_API_KEY and MAILGUN_DOMAIN in .env)
OAuth2 Outlook: Microsoft Outlook with OAuth2 authentication
Email Configuration Guide
SMTP Configuration

Set these environment variables in your .env file:

SMTP_SERVER=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASSWORD=your-app-password
SENDER_EMAIL=your-email@gmail.com
Resend Configuration

Set this environment variable in your .env file:

RESEND_API_KEY=re_xxxxxxxxxxxxx
SENDER_EMAIL=your-verified-sender@yourdomain.com

Note: You need to verify your sending domain in Resend dashboard first.

Mailgun Configuration

Set these environment variables in your .env file:

MAILGUN_API_KEY=your-api-key-xxxxxxxx
MAILGUN_DOMAIN=mg.yourdomain.com
MAILGUN_SENDER_EMAIL=noreply@mg.yourdomain.com

Note: Mailgun uses HTTP API (port 443), so it works on Render free tier. You need to verify your domain in Mailgun dashboard first.
