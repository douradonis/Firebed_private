Email Configuration Guide
Railway Proxy Configuration

For platforms that block SMTP (like Render free tier):

  1. Deploy the Railway email relay service from railway-email-relay/ directory
  2. Get the public URL from Railway (e.g., https://your-app.railway.app)
  3. Configure SMTP credentials in Railway environment variables or pass them in requests
  4. Enter the Railway URL above

Note: The Railway service still needs SMTP credentials to send emails.

SMTP Configuration

Set these environment variables in your .env file:

SMTP_SERVER=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASSWORD=your-app-password
SENDER_EMAIL=your-email@gmail.com
Resend Configuration

Set this environment variable in your .env file:

RESEND_API_KEY=re_xxxxxxxxxxxxx
SENDER_EMAIL=your-verified-sender@yourdomain.com

Note: You need to verify your sending domain in Resend dashboard first.
